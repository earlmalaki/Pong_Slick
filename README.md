# Pong_Slick

An iteration of basic PingPong game made using Slick2D library for Java.


Left racquet represents Player 1.

Right racquet represents Player 2.

Sound and power up to be implemented soon.

### Controls

Player 1:

  W - Move Up
  
  S - Move Down
  
  
Player 2:

  Arrow UP - Move Up
  
  Arrow Down - Move Down
